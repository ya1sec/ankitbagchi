Lately, I've been exploring the domain of Natural Language Processing and working on a miniature piece of software called KORPIS that aims to be a friend to poets. Heaps of text are generated by Markov Models trained on austere corpora, leaving poets to use the age-old cutup method to create works using the results. I'm writing a neverending poem called Impression of an Hour in collaboration with an experimental version of the program.
